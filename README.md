# QA Automation Assessment

This repository contains my QA Automation Assessment submission.

## Contents
- UI Automation (Cypress)
- API Automation (Postman)
- Theory Answers
- Coding Task

## UI Automation
The UI automation tests were implemented using **Cypress** following a **Page Object Model structure**.  
Test isolation was used to avoid repeated login and ensure stable test execution.  
Employee names are stored dynamically in a **JSON file** for reuse during the tests.

## API Automation
API tests were created using **Postman**, and the Postman collection is included in this repository.
